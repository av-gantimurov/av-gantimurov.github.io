Представляет собой ВПО семейства Mamont для устройств под управлением Android и предназначен для скрытного получения, отправки СМС, сбора информации о зараженном устройстве, отправки USSD команд, просмотра уведомлений и открытия произвольных URL.

При открытии приложение запрашивает доступ к получению и отправке SMS, к звонкам и камере.

![Пример запроса разрешений](Backdoor.Android.Mamont.12-20251120161017.png)
![Пример запроса разрешений](Backdoor.Android.Mamont.12-20251120161037.png)

После предоставления всех запрашиваемых разрешений открывается фишинговая страница, которая указана в настройках при создании приложения.

В данном случае открывается имитация сервиса по обмену изображениями. Такие ресурсы могут в свою очередь собирать учетные данные, которые введет пользователь.

![Пример фишинговой страницы](Backdoor.Android.Mamont.12-20251120161117.png)

![Пример формы по регистрации](Backdoor.Android.Mamont.12-20251120161204.png)

![Фишинговый сайт](Backdoor.Android.Mamont.12-20251120161307.png)

В других случаях может использоваться вполне легитимный сайт, чтобы не вызывать подозрения пользователя.

| Разрешение                                | Описание                                     |
| ----------------------------------------- | -------------------------------------------- |
| android.permission.SEND_SMS               | Отправка SMS                                 |
| android.permission.READ_PHONE_STATE       | Чтение состояние телефона (в том числе IMEI) |
| android.permission.READ_SMS               | Чтение SMS                                   |
| android.permission.RECEIVE_SMS            | Получение SMS                                |
| android.permission.READ_CONTACTS          | Чтение контактов                             |

## Конфигурация

Основные настройки для осуществления деятельности хранятся в классе `BuildConfig`

![Пример конфигурации](Backdoor.Android.Mamont.12-20251120165212.png)

В большинстве случаев она хранится в открытом виде.

Дополнительно на удаленный сайт отправляется информация о работе приложение через класс `RemoteLogger`. URL для отправки указан также в открытом виде.

![Пример RemoteLogging](Backdoor.Android.Mamont.12-20251120170122.png)

Кроме этого, в коде приложения реализован `EnhancedRemoteLogger`, который также может отправлять информацию, но он никак не используется.

![Пример EnhancedRemoteLogger](Backdoor.Android.Mamont.12-20251120165746.png)

## Команды

Всё взаимодействие с удаленным сервером злоумышленника происходить через `retrofit2`, команды и результаты работы отправляются в формате JSON.

Злоумышленник может отправлять следующие команды:

- `refresh_sms` - отправить SMS на удаленный сервер;
- `get_permissions_status` - отправить список доступных разрешений;
- `check_online` - сообщить о доступности через интернет;
- `send_ussd` - отправить USSD на указанные номер;
- `send_sms` - отправить SMS на указанный номер;
- `open_link` - открыть указанный URL во встроенном браузере;
- `delete_sms` - удалить указанную SMS.

## Особенности

В коде приложения есть отладочные комментарии и сообщения на русском языке.

![Пример сообщений](Backdoor.Android.Mamont.12-20251120171106.png)

В API для взаимодействия с удаленным сервером злоумышленника реализован метод `uploadVerificationData`, который в настоящее время никак не используется.

![Код uploadVerificationData](Backdoor.Android.Mamont.12-20251120171331.png)

Судя по коду, подразумевается, что будет отправлены сведения о номере паспорта, дате рождения и информация об устройстве.

Кроме этого, в строках приложения выявлены следующие неиспользуемые строки:

![Строки верфикации](Backdoor.Android.Mamont.12-20251120171743.png)

Что может свидетельствовать о том, что в дальнейшем в приложение будет добавлена возможность "пройти верификацию" у злоумышленника. Видеоматериалы с лицом пользователя могут использоваться для создания фишинговых сообщений и дальнейшего распространения.
